  • Este projeto elaborado por Diogo Gonçalo Varela Júlio, Diogo Silva Gonçalves e Luís Carlos Canelas Ferreira, no âmbito da disciplina de Programação Web-Cliente, consiste na implementação de um Website de cripto-moedas, no qual é possivel a visualização do TOP 100 das cripto-moedas e a possibilidade de consultar alguns detalhes da moeda. Neste Website é possivel adicionar e remover as cripto-moedas dos favoritos, visualizar alguns detalhes selecionando a cripto-moeda que se pretenda e tambem dispõe de uma barra de pesquisa que permite ao utilizador pesquisar a cripto-moeda que pretende em vez de pesquisar manualmente.
  • Para a elaboração deste Website, foi necessario utilizar as linguagens de programação como o HTML, CSS e JavaScript, tendo cada uma a sua função.
